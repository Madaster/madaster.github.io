## WELCHE INFORMATIONEN WERDEN HIER ANGEZEIGT?
Hier finden Sie alle Portfolios, die auf diesem Konto angelegt wurden. Ein Portfolio kann als eine spezifische Sammlung von Gebäuden betrachtet werden. Sie können die Ansicht der Portfolios ändern, indem Sie links auf "Ansicht" klicken. Hier können Sie zwischen einer Liste der Portfolios wählen, in denen der Name und der Eigentümer angezeigt werden oder Kacheln mit dem Bild des Portfolios. Sie können dies jederzeit ändern.

## WIE KANN ICH EIN NEUES PORTFOLIO ERSTELLEN?
* Klicken Sie auf "Portfolio hinzufügen".
* Geben Sie den Namen des Portfolios ein.
* Wählen Sie ein geeignetes Bild aus. Wenn Sie kein geeignetes Bild zur Verfügung haben, können Sie diesen Schritt überspringen. Sie können später ein Bild hinzufügen. In diesem Fall stellt das System ein Standardbild für das Portfolio zur Verfügung.
* Klicken Sie auf "Speichern". Das soeben erstellte Portfolio öffnet sich direkt und Sie können dieses Portfolio sofort nutzen.

## WIE KANN ICH EIN PORTFOLIO ANSEHEN?
Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten. Sie erhalten nun eine Übersicht über die Gebäude, die dem Gebäude hinzugefügt wurden, sowie eine Liste der Benutzer, die das Portfolio ansehen und bearbeiten können.

## WIE KANN ICH EIN PORTFOLIO LÖSCHEN?
Sie können ein Portfolio nur dann löschen, wenn es keine Gebäude mehr enthält. Was ist, wenn das Portfolio noch Gebäude enthält? Dann müssen Sie zuerst alle Gebäude aus dem Portfolio löschen, bevor Sie das Portfolio selbst löschen können. Weitere Informationen zum Löschen von Gebäuden finden Sie unter "Wie kann ich ein Gebäude aus einem Portfolio löschen". Nachdem Sie alle Gebäude aus dem Portfolio gelöscht haben, gehen Sie wie folgt vor:

* Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten.
* Scrollen Sie nach unten und klicken Sie auf "Portfolio anpassen".
* Scrollen Sie erneut nach unten und klicken Sie auf "Portfolio entfernen". Achtung: Sobald Sie ein Portfolio entfernt haben, können Sie die Aktion nicht mehr rückgängig machen!

## WIE KANN ICH DEN NAMEN ODER DAS BILD EINES PORTFOLIOS ÄNDERN?
* Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten.
* Scrollen Sie nach unten und klicken Sie auf "Portfolio anpassen".
* Geben Sie einen neuen Namen ein und/oder wählen Sie ein neues Bild aus.
* Klicken Sie auf "Speichern".

## WIE VIELE PORTFOLIOS KANN ICH ERSTELLEN?
Jeder organisatorische Benutzer, der berechtigt ist, ein Portfolio zu erstellen, kann beliebig viele Portfolios erstellen. Es gibt keine Begrenzung. Ein einzelner Benutzer kann ein einzelnes Portfolio erstellen, dieses Portfolio wird direkt bei der Registrierung für die Plattform erstellt.

## WER KANN AUF EIN PORTFOLIO ZUGREIFEN?
Der Ersteller eines Portfolios hat Zugriff auf das Portfolio. Dieser Benutzer ist der Kontoadministrator. Darüber hinaus kann dieser Benutzer anderen Benutzern Zugriff auf seine Portfolios gewähren. Weitere Informationen finden Sie im nachfolgenden Abschnitt.

## WIE KANN ICH BENUTZER ZU EINEM PORTFOLIO HINZUFÜGEN?
* Klicken Sie auf den Namen oder das Bild des Portfolios, um es zu öffnen.
* Gehen Sie zur Registerkarte "Benutzer".
* Hier sehen Sie eine Übersicht über die Benutzer, die auf das Portfolio zugreifen können.
* Klicken Sie auf "Benutzer hinzufügen".
* Geben Sie die E-Mail-Adresse des Benutzers ein, den Sie hinzufügen möchten. Wenn die Person, die Sie hinzufügen, bereits bei Madaster registriert ist, hat sie sofortigen Zugriff auf das Portfolio. Wenn die Person kein registrierter Benutzer ist, erhält sie eine E-Mail mit einer Anleitung zur Registrierung.
* Wählen Sie die Rolle, die Sie dem Benutzer zuweisen möchten, unter "Berechtigungen" aus und klicken Sie auf "Speichern". Im folgenden Abschnitt finden Sie weitere Informationen zu den verschiedenen Benutzerrollen.

## WELCHE ROLLEN KÖNNEN DEN BENUTZERN DER MADASTER-PLATTFORM ZUGEWIESEN WERDEN?
Benutzern der Madaster-Plattform können drei Rollen zugewiesen werden: Administrator, Manager und Leser. Die folgende Tabelle listet die Benutzerrechte der einzelnen Rollen auf.

Der Kontoadministrator, der auch der Eigentümer des Kontos ist, hat alle Berechtigungen. Als Administrator steuern Sie den Benutzerzugriff. Das bedeutet, dass Sie Benutzer hinzufügen und löschen sowie Benutzerrechte festlegen können. Darüber hinaus können Sie Portfolios erstellen, Gebäude, Materialien und Produkte in die Datenbank aufnehmen, Materialpässe exportieren, Dateien hochladen usw. Als Administrator können Sie Ihre eigenen Benutzerrechte nicht ändern.

Der Kontoadministrator kann Managerrechte vergeben. Als Manager können Sie die Portfolios und Gebäude, auf die Sie Zugriff erhalten haben, anpassen. Darüber hinaus kann den Benutzern Lesezugriff gewährt werden. Als Leser können Sie sich die Portfolios und Gebäude ansehen, zu denen Sie Zugang erhalten haben. Ein Leser kann auch Materialpässe herunterladen.

|------------------------------------------------------|---------------|---------|--------|
| Portfolios und darunterliegende Gebäude anzeigen     | Ja            | Ja      | Ja     |
| Portfolio erstellen                                  | Nein          | Nein    | Nein   |
| Portfolio und darunterliegende Gebäude anpassen      | Ja            | Ja      | Nein   |
| Portfolio und darunterliegende Gebäude löschen       | Ja            | Nein    | Nein   |
| Gebäude zum Portfolio hinzufügen                     | Ja            | Ja      | Nein   |

## WIE KANN ICH DIE ROLLEN VON BENUTZERN ÄNDERN, DIE ZUGRIFF AUF EIN PORTFOLIO HABEN?
* Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten.
* Gehen Sie zur Registerkarte "Benutzer" und klicken Sie auf "Berechtigungen bearbeiten".
* Ändern Sie die Rolle des betreffenden Benutzers.
* Klicken Sie auf "Speichern".

## WIE VIELE BENUTZER KANN ICH ZU MEINEM PORTFOLIO HINZUFÜGEN?
Dies hängt von Ihrem Madaster-Abonnement ab. Benutzer mit einem Einzelabonnement können keine Benutzer zu ihrem Konto hinzufügen. Bei Benutzern aus Unternehmen hängt dies von der Abonnementart ab.

## WIE KANN ICH BENUTZER LÖSCHEN?
* Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten.
* Gehen Sie zur Registerkarte "Benutzer". Dort sehen Sie eine Liste der Benutzer, die auf das Portfolio zugreifen können.
* Klicken Sie auf "Berechtigungen bearbeiten".
* Scrollen Sie erneut nach unten. Auf der rechten Seite jedes Benutzers befindet sich ein Papierkorbsymbol. Klicken Sie darauf, um den Benutzer zu löschen.
* Klicken Sie auf "Speichern". Nun kann der Benutzer nicht mehr auf dieses Portfolio zugreifen.

## WIE KANN ICH EIN GEBÄUDE ZU EINEM PORTFOLIO HINZUFÜGEN?
Ein Portfolio kann mehrere Gebäude enthalten.

* Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten.
* Klicken Sie auf die Schaltfläche "Gebäude hinzufügen".
* Geben Sie die erforderlichen Daten ein. Pflichtfelder sind mit einem Sternchen (*) gekennzeichnet.
* Klicken Sie auf "Speichern". Beachten Sie, dass das neue Gebäude in das Portfolio aufgenommen wurde. Es wird auch auf dem Startbildschirm angezeigt.

## WIE VIELE GEBÄUDE KANN ICH ZU EINEM EINZELNEN PORTFOLIO HINZUFÜGEN?
Jeder Nutzer kann beliebig viele Gebäude zu einem Portfolio hinzufügen. Es gibt keine Begrenzung, solange Sie die Grundfläche Ihres Abonnements nicht überschreiten. Weitere Informationen finden Sie im Abschnitt über die verschiedenen Abonnementtypen.

## WIE KANN ICH EIN GEBÄUDE AUS EINEM PORTFOLIO LÖSCHEN?
* Klicken Sie auf den Namen oder das Bild des Portfolios, das Sie öffnen möchten.
* Klicken Sie auf den Namen des Gebäudes, das Sie löschen möchten. Die Seite mit den Gebäudedetails wird geöffnet.
* Scrollen Sie nach unten und klicken Sie auf "Gebäude anpassen".
* Scrollen Sie erneut nach unten und klicken Sie auf "Gebäude entfernen". Nun erscheint eine Warnung, die Sie darauf aufmerksam macht, dass Sie mit dieser Aktion auch alle Dateien und Daten zu diesem Gebäude von der Madaster-Plattform löschen werden. 
Klicken Sie auf "Ja", wenn Sie sich dessen sicher sind. Das Gebäude ist nun entfernt. Achtung: Sie können diese Aktion nicht rückgängig machen.

## WIE KANN ICH MIT DER PORTFOLIOÜBERSICHT ZUR STARTSEITE ZURÜCKKEHREN?
Sie können jederzeit zur Startseite zurückkehren, indem Sie auf das Madaster-Logo in der linken oberen Ecke der Seite klicken, auf der Sie sich befinden. Sie können auch zur Portfolioübersicht zurückkehren, indem Sie in der Menüleiste auf "Portfolios" klicken. Sie können auch die Spurennavigation oben auf der Seite benutzen. Klicken Sie auf "Portfolio" oder "Startseite", um eine dieser Seiten zu öffnen.

## WIE KANN ICH MEIN PORTFOLIO AUF EINEN ANDEREN BENUTZER (Z. B. DEN NEUEN EIGENTÜMER DES GEBÄUDES) ÜBERTRAGEN?
Im Moment können Sie kein Portfolio oder Gebäude auf der Madaster-Plattform übertragen. Was Sie tun können, ist, den neuen Eigentümer als Benutzer hinzuzufügen, woraufhin Sie dem Benutzer "Administrator"-Rechte erteilen können. Dieser Benutzer ist nun auch der neue Eigentümer des Portfolios/Gebäudes.
